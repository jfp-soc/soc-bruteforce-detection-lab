# SOC Brute Force Detection Lab

## Objective
To detect and investigate a simulated SSH brute force attack using Linux authentication logs.

## Skills Demonstrated
- Log analysis
- Threat detection
- Incident reporting
- MITRE ATT&CK mapping

## Scenario
An internal Linux server experienced multiple failed SSH login attempts from an external IP address.

## Outcome
Identified brute force behavior and recommended remediation steps.
